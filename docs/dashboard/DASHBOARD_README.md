# Dashboard

The **canonical user guide** for the React app is:

**[../DASHBOARD_GUIDE.md](../DASHBOARD_GUIDE.md)** — data modes (LIVE / SIM), tabs, controls, metrics, exports.

## Quick run

```bash
# Terminal 1
python api.py

# Terminal 2
cd frontend && npm install && npm start
```

Open **http://localhost:3000** (proxy to the API is set in `frontend/package.json`).
